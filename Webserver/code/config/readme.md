# 命令解析  
提供个性化参数运行  
"p:m:o:s:t:l:g:"
* -p，自定义端口号
  * 默认9006
* -l，选择日志写入方式，默认同步写入
  * 0，同步写入
  * 1，异步写入
* -m，listenfd和connfd的模式组合，默认使用LT + LT
  * 0，表示使用LT + LT
  * 1，表示使用LT + ET
    * 2，表示使用ET + LT
    * 3，表示使用ET + ET
* -o，优雅关闭连接，默认不使用
  * 0，不使用
  * 1，使用
* -s，数据库连接数量
  * 默认为8
* -t，线程数量
  * 默认为8
* -g，日志写入等级，默认是1
  * 0，是debug
  * 1，是info
  * 2，是warn
  * 3，是error
